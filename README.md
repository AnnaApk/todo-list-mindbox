# ToDoList

![Общий вид ui](/public/image.png')

## React + TypeScript + Vite

### Запуск:
```
npm i
npm run start
```

### Запуск тестов:
```
npm run test
```
[Деплой приложения на Vercel](https://todo-list-mindbox-orcin.vercel.app/)

### Выполненный функционал:

* Поле для ввода новой задачи
* Списки задач:
  * общий
  * невыполненных задач
  * выполненных задач
* Количество оставшихся задач
* Очистка выполненных
